==========
Mouse Keys
==========

Mouse button key names available per platform.

Windows
=======

.. list-table::
   :header-rows: 1
   :widths: 40 60

   * - Key Name
     - Description
   * - ``mouse_left``
     - Left mouse button
   * - ``mouse_middle``
     - Middle mouse button (scroll wheel click)
   * - ``mouse_right``
     - Right mouse button
   * - ``mouse_x1``
     - Extra button 1 (back)
   * - ``mouse_x2``
     - Extra button 2 (forward)

   * - ``scroll_right``
