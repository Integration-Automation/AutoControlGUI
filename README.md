# je_auto_control

---

### JE Auto Control as a tool for GUI Testing 
#### Features
* Image Detect
* Keyboard Event
* Mouse Event
* Screen 
* Action file and executor
* Record Event
* CLI with action file
* timeout

```
# make sure you have install cmake libssl-dev
pip install je_auto_control
```

    * Python 3.5 & later
    * pip 19.3 & later


* Dev env
    * windows 11
    * osx 11 big sur
    * ubuntu 20.0.4


* Test on
    * windows 10 ~ 11
    * osx 10.5 ~ 11 big sur
    * ubuntu 20.0.4
    * raspberry pi 3B and 4B
